# Intellij IDEA

# IDEA简介

IDEA 全称 IntelliJ IDEA，是 Java 语言开发的集成环境，IntelliJ 在业界被公认为最好的 Java 开发工具之一。

# IDEA 小技巧

# 为类添加自动注释模板

IntelliJ IDEA -> Preferences... -> Editor -> File and Code Templates -> Includes -> 添加模板 -> Enable Live Templates -> Apply -> OK

类的注释模板

/**
* 
* @author ：liu jia
* @description：为类自动添加注释
* @date ：Created in 2021/4/28 19:55
* 
* /

此时，当你创建新的类时，就会自动生成注释了。


package com.tms;

/**
 * @author ：liu jia
 * @description：为类自动添加注释
 * @date ：Created in 2021/4/28 19:55
 */
public class HelloWorld {
 public static void main(String[] args) {
  System.out.println("Hello, world");
 }
}

# 为方法自动添加注释

IntelliJ IDEA -> Preferences... -> Editor -> Live Templates -> 加号 -> template Group

添加template Group, 编写任意group名称

选择OK

选择加号 -> Live template

设置快捷方式 * -> 描述输出模板的快捷方式 -> 设置模板

方法的注释模板

*
 * @Author liu jia
 * @Description
 * @Date $time$ $date$
 * @Param $params$
 * @return $return$
 **/

设置作用范围

编写template variables -> 选择OK

选择Apply -> 选择OK

此时，在方法上面 输入 /** 然后按Tab键，就会自动生成方法的注释。

public class HelloWorld {
 
 /**
  * @Author liu jia
  * @Description
  * @Date 23:30 2021/4/28
  * @Param [name, age]
  * @return java.lang.String
  **/
 public String userInfo(String name, int age) {
  return "用户信息如下：" + "姓名"+ name + "年龄" + age;
 }
 
}
