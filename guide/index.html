# 零基础学测试开发

# 什么是测试开发

简单来说就是指即会测试又会开发的专业技术人才.除了会制定测试计划,设计测试用例, 分析缺陷,参与程序代码的身材和产品设计的审查之外,还要会开发测试程序，提高测试效率。

# 为什么学测试开发

  • 为了提高测试效率
  • 为了自我提升

# 如何成为一名测试开发

  • 首先，有明确的学习目标。
  • 其次，根据实际情况选择适合自己的视频课程和书籍。
  • 最后，学以自用，独立解决实际问题。
