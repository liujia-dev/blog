# Spring概述

  • Spring框架实现了对bean的依赖管理及AOP的编程方式，降低了代码之间的耦合， 是一站式构建企业级应用的轻量级解决方案。
  • Spring框架是模块化的，允许开发自由选择需要使用的部分。
  • Spring框架支持声明式事务管理，通过RMI或Web服务远程访问用户的逻辑，并支持多种选择来持久化用户的数据
