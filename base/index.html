# 面向对象

# 面向对象的三个基本特征

# 封装

封装就是将类中的数据和方法隐藏起来,并且只让可信的类或对象操作,对不可信的类或对象进行信息隐藏.封装可以防止类的数据和方法被不可信的类或对象操作.

# 继承

继承就是子类继承父类的数据和方法,使得子类具有父类相同的行为. 继承有两种方式,即实现继承与接口继承.实现继承就是直接使用父类的属性和方法.接口继承就是只使用父类的属性和方法名称,由子类提供实现的能力.

# 多态

多态就是指一个类实例的相同方法在不同情形有不同表现形式.多态机制使得具有不同内部结构的对象可以共享相同的外部接口.最常见的多态就是将子类传入父类参数中,调用父类方法时根据传入的子类来决定具体的行为.

# 面向对象五大原则

  • 单一职责原则即一个类只做一件事

  • 开放封闭原则即对扩展开放,对修改封闭

  • 替换原则即子类必须能够替换其父类

  • 依赖倒置原则即依赖于抽象

  • 接口隔离原则即使用多个小的专门的接口
